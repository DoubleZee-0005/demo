# Wedding Invitation

A beautiful, interactive wedding invitation featuring:
- **Scratch-to-reveal** glitter heart effect.
- **Glitter texture** and elegant animations.
- **Responsive design** optimized for mobile and desktop.
- **Background music** that plays on user interaction.
- **Add to Calendar** functionality.

## Preview
This invitation is designed to be hosted on GitHub Pages.

## Tech Stack
- HTML5 / CSS3 (Vanilla)
- Tailwind CSS (via CDN)
- Canvas API (for the scratch effect)
- Confetti.js

## Local Setup
1. Clone the repository.
2. Open `index.html` in your browser.
